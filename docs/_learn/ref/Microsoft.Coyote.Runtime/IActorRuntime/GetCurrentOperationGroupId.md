# IActorRuntime.GetCurrentOperationGroupId method

Returns the operation group id of the actor with the specified id. Returns Empty if the id is not set, or if the [`ActorId`](../../Microsoft.Coyote.Actors/ActorIdType) is not associated with this runtime. During testing, the runtime asserts that the specified actor is currently executing.

```csharp
public Guid GetCurrentOperationGroupId(ActorId currentActorId)
```

| parameter | description |
| --- | --- |
| currentActorId | The id of the currently executing actor. |

## Return Value

The unique identifier.
