# StateMachine.Transition.Type enumeration

Defines the type of a [`StateMachine`](StateMachineType) transition.

## Values

| name | value | description |
| --- | --- | --- |
| None | `0` | A transition that does not change the [`State`](StateMachineStateType). This is the value used by [`None`](StateMachine.Transition/None). |

| PopState | `4` | A transition created by [`PopState`](StateMachine/PopState) that pops the current [`State`](StateMachineStateType) from the stack of [`StateMachine`](StateMachineType) states. |
| Halt | `5` | A transition created by [`Halt`](StateMachine/Halt) that halts the [`StateMachine`](StateMachineType). |
