# Actor.DeferEvent method

Sets the actor to defer all events of the the specified type. This can be reverted by setting the *defer* parameter to false.

```csharp
protected void DeferEvent(Type eventType, bool defer = true)
```

| parameter | description |
| --- | --- |
| eventType | The event type to defer. |
| defer | True to defer events of the specified type, else false. |
