# Actor.IgnoreEvent method

Sets the actor to ignore all events of the the specified type. This can be reverted by setting the *ignore* parameter to false.

```csharp
protected void IgnoreEvent(Type eventType, bool ignore = true)
```

| parameter | description |
| --- | --- |
| eventType | The event type to ignore. |
| ignore | True to ignore events of the specified type, else false. |
