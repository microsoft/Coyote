# TaskExtensions.WrapInControlledTask method (1 of 2)

Returns a dummy controlled [`Task`](../Task.md) that wraps this uncontrolled Task.

```csharp
public static Task WrapInControlledTask(this Task @this)
```

## Remarks

The returned dummy controlled [`Task`](../Task.md) does not actually take control of the uncontrolled Task during systematic testing, so this method should only be used to cross an interface boundary where a controlled [`Task`](../Task.md) must be temporarily converted into an uncontrolled Task and then coverted back to a controlled [`Task`](../Task.md).
